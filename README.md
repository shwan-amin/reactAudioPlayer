reactAudioPlayer is a web-application that allows a user to control audio tracks from a pre-set 3 track list. It has functionality such as play,
pause, skip and skip-back the songs. This project uses self-made components to render the card sections and buttons.

This project helped me practice:
- React context
- React project heirarchy and design
- Components design
- Handling props and children.
- useState and useEffect.
